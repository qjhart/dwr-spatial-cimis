## DESCRIPTION ##

v.in.et retrieves ETo station data from DWR's et.water.ca.gov website. 


## NOTES ##

The total daily insolation, G, is used to compute the daily reference
  evapotranspiration (ET0) using the FAO Penman-Monteith method.
